# Job-Seeker-App
A small website for job seekers

```
git clone https://github.com/Jigyasa08/Job-Seeker-App.git

/
npm install

/backend/AuthAPI
nodemon server.js

/backend/UserAPI
nodemon server.js

/frontend
npm install
npm run start

```
